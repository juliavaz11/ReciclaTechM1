# ♻️ ReciclaTech - Plataforma de Doação de Eletrônicos

O **ReciclaTech** é uma landing page desenvolvida com **HTML e CSS** que incentiva o descarte consciente e a reutilização de equipamentos eletrônicos. A proposta da plataforma é conectar pessoas que desejam doar dispositivos em bom estado com quem precisa deles, promovendo inclusão digital e sustentabilidade.

---

## 📖 Sobre o projeto

O ReciclaTech foi criado para conscientizar sobre a importância da reciclagem e do reaproveitamento de eletrônicos, reduzindo o descarte inadequado e ampliando o acesso à tecnologia.

A plataforma apresenta informações sobre o funcionamento do serviço, produtos disponíveis para doação, depoimentos de usuários e chamadas para ação, incentivando a participação da comunidade.

---

## ✨ Funcionalidades

- Landing page responsiva.
- Apresentação da proposta da plataforma.
- Seção explicando como funciona o processo de doação e aquisição.
- Exibição de estatísticas da plataforma.
- Cards com eletrônicos disponíveis.
- Depoimentos de usuários.
- Botões de chamada para ação (Call to Action).
- Rodapé com informações institucionais e redes sociais.

## 🚀 Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/reciclatech.git
```

Entre na pasta do projeto:

```bash
cd reciclatech
```

Abra o arquivo `index.html` em qualquer navegador ou utilize a extensão **Live Server** no Visual Studio Code.
